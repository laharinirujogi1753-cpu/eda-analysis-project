# Exploratory Data Analysis Project

## Objective
To perform exploratory data analysis on the Titanic dataset and uncover insights using Python.

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## Dataset
Titanic dataset

## Analysis Performed
- Data summary and statistics
- Missing values check
- Survival count analysis
- Survival by gender
- Age distribution
- Correlation heatmap

## Key Findings
- More passengers did not survive than survived.
- Female passengers had a higher survival rate.
- Most passengers were between 20 and 40 years old.
- Missing values existed in age and deck columns.

## Conclusion
EDA helps understand patterns, detect issues, and prepare data for machine learning.
